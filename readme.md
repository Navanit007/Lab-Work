3. Theory / Background

- Discord supports real-time communication, file sharing, and community building, making it a valuable tool for project collaboration.

4. Apparatus / Tools Used

- Software: GitHub Desktop / Git CLI, Discord application, multimedia presentation tools.
- Network: Internet connection for GitHub and Discord access.
